🎵 Harmonia CORS 代理

专为音乐播放器设计的轻量级跨域代理 · 基于 Vercel Edge Runtime
稳定 · 私密 · 支持网易云 / 通用 API

🌐 代理地址

https://cors.harmoniamusicplayer.dpdns.org/api/proxy

支持 两种调用模式：通用代理（任何URL） 和 特定音乐API模式（参数映射）。

🔀 通用模式 · 使用 ?url= 🎯 音乐模式 · 使用 types source id 等

🔗 模式一 · 通用 CORS 代理

通过 url 参数传递任意目标地址，代理将直接转发（保留响应类型）。
适合请求网易云歌词、图片或其他任意 API。

参数说明

参数类型描述
url string (必填) 要代理的完整 URL，必须经过 encodeURIComponent 编码。

示例

// 获取网易云歌词 (ID: 3325884349)
const target = 'https://music.163.com/api/song/lyric?id=3325884349&lv=1';
const proxyUrl = `https://cors.harmoniamusicplayer.dpdns.org/api/proxy?url=${encodeURIComponent(target)}`;

fetch(proxyUrl)
  .then(res => res.json())
  .then(lyric => console.log(lyric.lrc.lyric));
// 直接浏览器测试（已编码）

🎼 模式二 · 音乐数据源代理

固定转发到 https://music-api.gdstudio.xyz/api.php，支持该源的所有参数。
适合搜索、获取播放链接、专辑封面等。

支持参数

参数类型说明
typesstring请求类型：search, lyric, pic, url 等
sourcestring数据源：netease, tencent, xiami 等
idstring歌曲/歌单 ID（用于 lyric, pic, url）
namestring搜索关键词（当 types=search 时）
pagesnumber页码，默认 1
countnumber每页数量，默认 20
brnumber码率：128, 320, 999（用于 types=url）
sizenumber图片尺寸（用于 types=pic）

示例

// 搜索周杰伦 (网易云)
fetch('https://cors.harmoniamusicplayer.dpdns.org/api/proxy?types=search&source=netease&name=周杰伦&count=5')
  .then(res => res.json())
  .then(songs => console.log(songs));
// 获取歌曲播放地址
fetch('https://cors.harmoniamusicplayer.dpdns.org/api/proxy?types=url&source=netease&id=3325884349&br=320')
  .then(res => res.json())
  .then(data => console.log(data.url));

⚠️ 注意事项

  • URL 编码：使用通用模式时，url 参数必须用 encodeURIComponent() 编码，否则可能解析失败。
  • 请求头：代理已内置浏览器级 User-Agent 及 Referer，可正常通过网易云等接口校验。
  • 响应大小：Edge Function 对响应体有 4MB 限制，适合歌词、封面等，大文件请谨慎。
  • 仅支持 GET：目前仅允许 GET 请求，OPTIONS 用于 CORS 预检。
  • 使用限制：此代理为个人项目优化，请勿用于大规模爬取，合理使用。
📌 关于逐字歌词
网易云逐字歌词需添加 &yv=1 参数，例如：
https://music.163.com/api/song/lyric?id=3325884349&lv=1&yv=1
通过通用模式传入即可。若返回数据缺少 lrc，通常是 User-Agent 问题（本代理已修复）。
